# Hello World

This is the first post on this blog. Nothing groundbreaking — just making sure the pipeline works.

## The Stack

This blog runs on:

- **Next.js 13** with Pages Router
- **Velite** for content management (markdown → typed data)
- **Tailwind CSS** for styling
- **GitHub Pages** for hosting with automatic deployment

Write a `.mdx` file in `content/posts/`, push to `main`, and it goes live in about two minutes. No CMS, no database, no fuss.

More posts, probably. About things I'm building, things I'm learning, and things that break in interesting ways.
